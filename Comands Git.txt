Comands Git
1º - Como subir um arquivo no GitHub
cd Documents
cd diretory
dir
git init
git add .
git commit -m "first commit"
git remote add origin [link do repositório]
git push -u origin master
2º - Como atualizar repositório no GitHub
cd Documents
cd diretory
dir
git status
git add .
git commit -m "second commit"
git push -u origin master
